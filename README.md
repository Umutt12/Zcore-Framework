# 🌌 Zcore-Framework v2.5
> **Advanced Network Recon & Security Auditing Tool**

Zcore is a modular framework built for Termux, focusing on network discovery, vulnerability analysis, and security auditing with a "Yofukashi no Uta" aesthetic.

---

## 🛠 Features
- 🔍 **Smart Discovery:** Automatically detects network ranges and bypasses Android restrictions.
- 📦 **Recon Suite:** Detailed service scanning, OS detection, and full port audits.
- 🛡️ **Vuln Scan:** Integrated NSE (Nmap Scripting Engine) for security checks.
- 🕸️ **Web Audit:** Locates hidden admin panels and login pages.
- 🦊 **Scam Hunter:** Specialized module for social media security auditing.

```bash
pkg update && pkg install git nmap curl -y
